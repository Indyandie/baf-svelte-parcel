## Quickstart


## Install package globally

In your terminal

```bash
npm i -g baf-svelte-parcel
```

## Create baf-svelte-parcel

Run

```bash
   basp # will clone and install in /barp if it doesn't already exist
```

## Use custom name

```bash
   basp your-app-name
```
